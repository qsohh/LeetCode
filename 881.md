## Boats to Save People

Two pointers.

```python
class Solution:
    def numRescueBoats(self, people: List[int], limit: int) -> int:
        people.sort(reverse=1)
        p1, p2, res = 0, len(people)-1, 0
        while p1 <= p2:
            if people[p1] + people[p2] <= limit:
                p1 += 1
                p2 -= 1
            else:
                p1 += 1
            res += 1
        return res
```
