## Find Minimum in Rotated Sorted Array

鉴于如果数组被转过了的话，```nums[0]```一定会大于```nums[-1]```，那就很容易二分了：

```python
class Solution:
    def findMin(self, nums: List[int]) -> int:
        if nums[0] < nums[-1]:
            return nums[0]
        p1, p2 = 0, len(nums) - 1
        while p1 < p2:
            p = (p1 + p2)//2
            if nums[p] < nums[p2]:
                p2 = p
            else:
                p1 = p + 1
        return nums[p1]
```
