## Maximum Number of Vowels in a Substring of Given Length

```python
class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        nb_c = len(s)
        vowels = set(['a', 'e', 'i', 'o', 'u'])
        cur, res = 0, 0
        for ele in s[0:k]:
            if ele in vowels:
                cur += 1
                res += 1
        for i in range(0, nb_c-k):
            cur += -int(s[i] in vowels) + int(s[i+k] in vowels)
            if cur > res:
                res = cur
        return res
```
