## Array Nesting

检测环。但是我也没想明白为甚么我写的这个运行得那么慢

```python
class Solution:
    def arrayNesting(self, nums: List[int]) -> int:
        res = 1
        for i, n in enumerate(nums):
            if n < 0:
                continue
            ct, nums[i], cur = 1, -1, n
            while nums[cur] >= 0:
                ct += 1
                temp = nums[cur]
                nums[cur] = -1
                cur = temp
            res = max(res, ct)
        return res
```
