## Merge Strings Alternately

```python
class Solution:
    def mergeAlternately(self, word1: str, word2: str) -> str:
        p, l1, l2 = 0, len(word1), len(word2)
        res = ''
        while p < l1 and p < l2:
            res += word1[p] + word2[p]
            p += 1
        if l1 > p:
            return res + word1[p:]
        if l2 > p:
            return res + word2[p:]
        return res
```

看看大佬的版本：
```python
from itertools import zip_longest
class Solution:
    def mergeAlternately(self, word1: str, word2: str) -> str:
        return "".join([a + b for a,b in list(zip_longest(word1,word2, fillvalue=""))])
```
