## Simplify Path

```python
class Solution:
    def simplifyPath(self, path: str) -> str:
        stack = []
        dirs = path.split('/')
        cur = ''
        for directory in dirs:
            if directory:
                if directory == '..':
                    if stack:
                        stack.pop()
                elif directory != '.':
                    stack.append(directory)
        return '/'+'/'.join(stack)
```
