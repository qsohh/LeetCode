## Count Good Nodes in Binary Tree

正常来讲，DFS同时记下来当前路径上最大值就好了，如下 ：

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def goodNodes(self, root: TreeNode) -> int:
        if not root:
            return 0
        def dfs(node, curmax):
            if not node:
                return 0
            if curmax <= node.val:
                return dfs(node.left, node.val) + dfs(node.right, node.val) + 1
            return dfs(node.left, curmax) + dfs(node.right, curmax)
        return dfs(root, root.val)
```

看了一个大佬的代码，里边用了```collections.filter```，感觉好像很厉害的样子 ：

```python
class Solution:
    def __init__(self):
        self.count = 0

    def goodNodes(self, root):
        for child in filter(None, [root.left, root.right]):
            self.count += child.val >= root.val
            child.val = max(child.val, root.val)
            self.goodNodes(child)
  
        return self.count + 1
```
