## Best Time to Buy and Sell Stock using Strategy

貌似用prefix算得会更快点

```python
class Solution:
    def maxProfit(self, prices: List[int], strategy: List[int], k: int) -> int:
        l = len(prices)
        res = sum([prices[i]*strategy[i] for i in range(l)])
        cur = sum([prices[i]*strategy[i] for i in range(k//2, l)])
        cur += + sum([prices[i]*(1-strategy[i]) for i in range(k//2, k)])
        res = max(res, cur)
        if l == k:
            return res
        for i in range(k, l):
            cur += prices[i-k]*strategy[i-k] - prices[i-k//2] + prices[i]*(1 - strategy[i])
            res = max(res, cur)
        return res
```
