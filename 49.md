## Group Anagrams

“万物皆可直接 hash”理论的实践 ：可是这个实际效果跑出来非常慢

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        def hsword(word):
            res = [0] * 26
            st = ""
            for c in word:
                res[ord(c)-97] += 1
            for ct in res:
                st += "{:03}".format(ct)
            return st
        res, inds = [], dict()
        for word in strs:
            hs = hsword(word)
            if hs in inds:
                res[inds[hs]].append(word)
            else:
                inds[hs] = len(res)
                res.append([word])
        return res
```

但是实际上，把```list```直接转换成```tuple```不就得了 ：

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        res = defaultdict(list)
        for word in strs:
            ct = [0] * 26
            for s in word:
                ct[ord(s)-97] += 1
            res[tuple(ct)].append(word)
        return res.values()
```

大佬们用```sorted```之后实际还能更快点 ：

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        res = defaultdict(list)
        for word in strs:
            res["".join(sorted(word))].append(word)
        return res.values()
```
