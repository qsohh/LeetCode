## Flip String to Monotone Increasing

```python
class Solution:
    def minFlipsMonoIncr(self, s: str) -> int:
        res, n1 = 0, 0
        try:
            si = s.index('1')
        except ValueError:
            return 0
        for c in s[si:]:
            if c == '0':
                res = min(res+1, n1)
            else:
                n1 += 1
        return res
```
