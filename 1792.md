## Maximum Average Pass Ratio

基础最小堆。

```python
class Solution:
    def maxAverageRatio(self, classes: List[List[int]], extraStudents: int) -> float:
        if len(classes) == 1:
            return (classes[0][0] + extraStudents) / (classes[0][1] + extraStudents)
        hq = []
        for p, t in classes:
            heapq.heappush(hq, [(p-t)/(t*t+t), p, t])
        for _ in range(extraStudents):
            d, p, t = heapq.heappop(hq)
            heapq.heappush(hq, [(p-t)/(t*t+3*t+2), p+1, t+1])
        return sum(p/t for d, p, t in hq)/len(hq)
```
