## Subsets II

不知道用```pop```的话会不会比```for```快一点

```python
class Solution:
    def subsetsWithDup(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        res = [[]]
        for nb in nums:
            res_2 = res.copy()
            for ele in res:
                if ele+[nb] not in res:
                    res_2.append(ele+[nb])
            res = res_2.copy()
        return res
```
